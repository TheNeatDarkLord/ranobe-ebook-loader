# ranobe-ebook-loader

Юзерскрипт для скачивания книг (fb2, epub) c https://ranobes.com/ , https://ранобэ.рф/ , https://tl.rulate.ru/ , https://jaomix.ru/

> **Это форк** [оригинала Taraflex](https://github.com/Taraflex/ranobe-ebook-loader) с починкой под новую защиту и вёрстку ranobes.com (2025-2026). Оригинал перестал работать; здесь восстановлена работоспособность и добавлена докачка/автопродолжение. Лицензия MIT, авторство оригинала сохранено.

## 📖 Как пользоваться (пошагово)

### Шаг 1. Поставить расширение для юзерскриптов (один раз)

Нужно любое из:
- [**Tampermonkey**](https://tampermonkey.net/) — самый популярный, подходит для Chrome / Edge / Firefox / Opera;
- [**Violentmonkey**](https://violentmonkey.github.io/get-it/) — рекомендуется, не спрашивает разрешения на загрузку картинок;
- [Greasemonkey](https://www.greasespot.net/) — для Firefox.

Установите расширение из магазина расширений вашего браузера.

### Шаг 2. Установить сам скрипт (один раз)

1. Откройте ссылку: [**Нажать для установки юзерскрипта**](https://raw.githubusercontent.com/TheNeatDarkLord/ranobe-ebook-loader/master/build/ranobe-ebook-loader.user.js)
2. Расширение покажет страницу установки → нажмите **«Установить» / «Install»**.
3. Если раньше стояла старая версия — сначала удалите её в панели расширения, потом ставьте эту.

Проверить, что встало: откройте `F12 → Console`, там при загрузке страницы будет строка вроде `ranobe-ebook-loader 0.5.x`.

### Шаг 3. Скачать книгу

1. Откройте страницу новеллы на `ranobes.com` (адрес вида `ranobes.com/ranobe/...`).
2. Дождитесь полной загрузки. Если появится проверка Cloudflare / «Я не робот» — **пройдите её**.
3. В блоке «Рейтинг новеллы» появятся кнопки **«Скачать *.FB2»** и **«Скачать *.EPUB»**.
4. Нажмите нужную **один раз**. По центру экрана появится шкала с процентами — значит идёт загрузка.
5. **Не закрывайте и не трогайте вкладку**, пока идёт скачивание. По завершении файл сохранится сам.

> EPUB можно скачать и горячей клавишей <kbd>CTRL</kbd>+<kbd>S</kbd>.

### Шаг 4. Большие книги — докачка заходами

Сайт защищён антиботом и пропускает примерно **~30 глав за одно прохождение проверки**, потом временно блокирует (это лимит по числу запросов, а не по скорости; ожидание блок не снимает — нужно заново пройти проверку «Я не робот»).

- **Короткая новелла (до ~30 глав)** — скачается за один клик, целиком.
- **Большая книга** качается так:
  1. нажали «Скачать» → набралось ~30 глав → блокировка;
  2. скрипт **сам сохраняет прогресс и перезагружает страницу**;
  3. появилась «Я не робот» — **пройдите её один раз**;
  4. скрипт **сам продолжает с того места**, где остановился (уже скачанные главы не качаются заново);
  5. так повторяется, пока книга не докачается полностью → файл сохранится сам.

Прогресс хранится в браузере, поэтому даже если закрыли вкладку — при следующем запуске докачка продолжится, а не начнётся заново.

### Если что-то не так

- **Сразу пишет блокировку на 0%** — ваш IP временно в бане из-за частых попыток. **Подождите 20–30 минут, не заходя на сайт**, затем повторите.
- **Кнопок «Скачать» нет** — страница ещё не прогрузилась или висит проверка Cloudflare; обновите и пройдите её.
- **Хотите остановить докачку** — нажмите «Cancel» на шкале загрузки или просто закройте вкладку.

## Что исправлено и добавлено в этом форке

- **Адаптация под новую вёрстку ranobes.com** — сайт убрал микроразметку (`itemprop`), из-за чего скрипт падал с `Cannot read properties of null`. Метаданные (обложка, автор, жанры, описание, год) теперь берутся из актуальных селекторов и Open Graph.
- **Исправлены пустые книги** — главы сохранялись только с заголовком, без текста (шаблоны читали поле `data`, а главы хранят текст в `text`). Теперь тело глав сохраняется в FB2 и EPUB.
- **Обход антибота** — главы грузятся последовательно с человеческой паузой (вместо 5 параллельных запросов, которые сразу триггерили защиту).
- **Детект блокировок** — распознаются страницы-заглушки Cloudflare и «Антибот»; вместо краша/пустого файла показывается понятное сообщение, а пустые (soft-blocked) главы перезапрашиваются.
- **Докачка с места обрыва** — скачанные главы и оглавление кэшируются в IndexedDB; повторный запуск продолжает, а не начинает заново.
- **Автопродолжение** — при блокировке скрипт сохраняет прогресс, перезагружает страницу и сам продолжает скачивание после прохождения проверки.
- **Видимая шкала прогресса** — исправлен краш при монтировании (`appendChild` на `null`), из-за которого не появлялась полоса загрузки.
- **Поддержка зеркал** `ranobes.top` и `ranobes.net` в дополнение к `ranobes.com`.
